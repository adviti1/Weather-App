## ğŸ“¦ Local Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/weather-app.git
   cd weather-app
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Local Env**
   Create a `.env.local` file:
   ```env
   API_KEY=your_gemini_api_key_here
   ```

4. **Run**
   ```bash
   npm run dev
   ```
